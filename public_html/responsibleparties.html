  • Dr. Paul Baran was hired by the government to figure out the problem at hand.
  • Baran and his team started with trying to find a way to make sure data always made it to where it needed to get to even with distruption or being destroyed.
  • Baran created the “Hot Potato Routing” meant a message transiting a network got to where it was going even if part of a network was destroyed.
  • Baran was inspired by Claude Shannon’s 1953 mouse in a maze model. He believed if we could model a mouse going through a maze, we should be able to model same for information. The information flows along in each of the packets that’s necessary to find its way through the network, such like mice in a maze finding their way.
  • From this idea, Baran also included the methodology of the telegraph for this idea – Baran decided to chop up each message into small equal sized pieces (packets). Each packet has a “to” and “from” address. Once a header has been put on a packet indicating where the packet is to go, the packet goes through the network with the other packets following behind. The packets find their way through paths that are available to get to where they need to go.
  • Baran then decided to let a node decide where to send each packet while going through the network on its way to the recipient.
  • A node would keep copies of the packets and keep sending packets out until it reached its destination. This process was called network switching.
  • Network switching became better than the telephone system because information always got through as long as a part of the network functioned.

  • 1966 – Bob Taylor, ARPA Scientist needed a separate terminal to log onto each timeshared computer across the country. Since no 2 computers could communicate with each other, Taylor had to move through each terminal. He figured a single terminal would be better if it could communicate with all the computers at once instead through one network.
  • Taylor asked for money to build a computer network – “resource sharing” to save time and money for ARPA.
  • Taylor recruited Dr. Lawrence (Larry) Roberts to help with his project. Roberts had built the world’s first experimental connections between 2 computers at MIT.
  • Robert had to figure out how to network main frame computers.
  • An engineer inspired Larry by suggesting building identical special purposes mini computers to form the network and tie each timeshared computer to them.
